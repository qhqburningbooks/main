# GHQ焚書図書補完計画

著作権切れしたGHQ焚書図書を電子書籍化して公開するプロジェクトです。

## GHQ焚書図書とは

戦後、GHQにより戦前・戦中の書物が7,000点以上没収されました。
これらの書物を補完することで、戦後に失われた戦前・戦中の日本の言論空間を掘り起こし、日本の伝統と精神を取り戻しましょう。
